# Advanced Excel Business Analytics

## Overview

This repository contains Excel-based business analytics projects completed to strengthen data analysis, reporting, and spreadsheet automation skills. The projects cover core Excel functions, data manipulation techniques, and business reporting scenarios commonly used in MIS, Reporting Analyst, and Data Analyst roles.

---

## Skills Demonstrated

* Advanced Excel
* VLOOKUP
* INDEX & MATCH
* IF, AND, OR Functions
* SUMIFS, COUNTIFS, AVERAGEIFS
* Date & Time Functions
* Sorting & Filtering
* Data Analysis
* Data Cleaning
* Charts & Data Visualization
* Business Reporting

---

## Project Modules

### Arithmetic Functions

* Employee and salary analysis
* Department-wise calculations
* Aggregate functions and conditional calculations

### Logical Functions

* Employee classification
* Eligibility and bonus calculations
* Business rule implementation using IF, AND, and OR

### Lookup Functions

* Data retrieval using VLOOKUP
* Dynamic lookups using INDEX & MATCH
* Error handling using IFERROR

### Date Functions

* Age calculations
* Date extraction and formatting
* Current date and time functions

### Sorting & Filtering

* Multi-level sorting
* Custom sorting
* Text and date filtering

### Data Visualization

* Charts and reports for business insights
* Trend and performance analysis
